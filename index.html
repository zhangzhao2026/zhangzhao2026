    XiZhou's Life System¶

    人生目标¶

    1. 赚一千万。
    2. 在一线城市买两套房。
    3. 送我姐一套，把户口都迁过去，后半生做邻居。
    4. 找全国最好的老师，学投资、学写作、学思维、学国学。

    2026年目标¶

    1. 把自己调整到人生最佳状态，养成三个好习惯：规律作息、阅读、思考。
    2. 活下去，把电商做好。目标：每月盈利满足日常开销，买一台新电脑，年底手里留有存款。
    3. 精通一条AI赛道，精通一门AI技术。

    工作伙伴¶

    电脑¶

    软件¶

    1. VS Code：写博客；
    2. Obsidian：
      • 日记库：写日记、记（项目/阅读/其他）笔记；
      • 知识库：将上述笔记存入结构化的文档系统；
      • 阅读库：存文章、存书籍，渐进阅读和挖空复习；
    3. Vivaldi：效率极高的浏览器；
    4. Anki：制作闪卡。

    桌面¶

    电脑端创建三个桌面（工作/生活/学习），将不同的软件分散放置，预设方案参考如下：

    1. 工作
      • Obsidian 知识库
      • Vivaldi
    2. 生活
      • 左右分屏：VS Code 博客、Obsidian 日记库
      • Vivaldi/Microsoft Edge
      • Microsoft To Do
      • 网易云音乐
    3. 学习
      • Obsidian 阅读库
      • Anki
      • ima

    手机¶

    软件¶

    1. 电商类APP：赚钱；
    2. 习惯类APP
      1. iBetter：培养习惯；
      2. aTimeLogger：记录时间；
      3. Microsoft To Do：做每日计划；
      4. AnkiDroid：培养长期记忆；
      5. 词根单词：学一门外语，锻炼思维；
      6. 相机：记录当下时刻。
    3. 资讯类APP：获取信息差；
    4. 网易云APP：娱乐放松。
